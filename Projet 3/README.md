- # Projet 03 - Requêtez une base de données avec SQL 

Projet avec une liste d'exercices guidé pour effectuer mes premières requêtes de base de données avec SQL 
- Découvrez les différents types de données
- Découvrez la conception de schéma relationnel
- Découvrez la création et le chargement d'une base de données
- Découvrez la rédaction de requêtes SQL
- Exercez-vous à la rédaction de requêtes SQL
- Rédigez un support de présentation de votre méthodologie

- # Projet 5 - Créez et utilisez une base de données immoblière avec SQL

## Scénario :
Je viens d’être embauché(e) chez Laplace Immo, un réseau national d’agences immobilières qui lance un projet stratégique interne nommé DATAImmo. L’objectif de l’entreprise est de mieux anticiper les prix de vente grâce à l’exploitation des données, afin de se différencier sur le marché. La CTO me confie une mission clé : faire évoluer la base de données qui centralise les transactions immobilières et foncières en France, à partir de plusieurs sources (DVF, INSEE, référentiel géographique de data.gouv). Je dois préparer mes livrables pour une réunion de validation, puis, une fois la conception approuvée, passer à l’implémentation opérationnelle de la base.
